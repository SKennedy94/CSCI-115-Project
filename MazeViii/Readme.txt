===== README =====
Submitted in a project folder called Maze. 
Under this folder there is an executable that launches the game. 
You may also open the Maze CBP file and compile and run the project in CodeBlocks 

(Compilled using C++ 14 standards)



===== CONTENTS ======
Common - contains the dll files for OpenGL also relevent dll files in main directory
images - contains the images that are loaded into the game
include - contains the .h files 
levels - contains text files used to load the levels and their elements
sounds - contains the music/sound effect (.mp3 files)
src - contains the .cpp files
